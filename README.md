# Webinar Funnel Research

This project focuses on how webinar funnels are built from zero and how effective of a marketing tool it is, I have sourced from 10 real people across different fields from webinar-craft specialists to B2B SaaS operators. These examples are going to help us understand different point of view's when comes to organising a successful webinar, one that actually convert into leads and know that only a topic does not make a webinar interesting a lot more goes into it.
